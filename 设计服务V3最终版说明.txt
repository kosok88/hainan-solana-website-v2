## 💡 协议关联的技术实现

### 方式1：localStorage（当前实现）

**优点：**
- ✅ 简单快速
- ✅ 无需后端
- ✅ 即时保存

**缺点：**
- ❌ 仅在本地浏览器
- ❌ 清除缓存会丢失
- ❌ 不能跨设备

**适合场景：**
- 小型业务
- 快速上线
- 演示展示

**代码示例：**
```javascript
// 注册时保存
localStorage.setItem('regName', '张三');
localStorage.setItem('regPhone', '13800138000');
localStorage.setItem('regEmail', 'zhang@example.com');

// 协议页读取
const name = localStorage.getItem('regName');
const phone = localStorage.getItem('regPhone');
const email = localStorage.getItem('regEmail');

// 签署后保存
localStorage.setItem('agreement_DS20241225001', JSON.stringify({
  orderNo: 'DS20241225001',
  userName: name,
  ...
}));
```

**如果需要：**
- ✅ 跨设备访问

**需要搭建：**
1. 后端服务器（Node.js / Python / PHP）
2. 数据库（MySQL / MongoDB）
3. API接口

**数据库表结构示例：**

**用户表（users）**
```sql
CREATE TABLE users (
  id INT PRIMARY KEY AUTO_INCREMENT,
  name VARCHAR(100),
  phone VARCHAR(20),
  email VARCHAR(100),
  created_at DATETIME
);
```

**协议表（agreements）**
```sql
CREATE TABLE agreements (
  id INT PRIMARY KEY AUTO_INCREMENT,
  order_no VARCHAR(50) UNIQUE,
  user_id INT,
  service_type VARCHAR(100),
  service_level VARCHAR(50),
  total_fee VARCHAR(50),
  deposit_paid BOOLEAN DEFAULT FALSE,
  balance_paid BOOLEAN DEFAULT FALSE,
  agreed_date DATETIME,
  status VARCHAR(20)
);
```

**API端点：**
```
POST /api/register - 用户注册
POST /api/agreement/create - 创建协议
GET /api/agreement/:orderNo - 查询协议
PUT /api/agreement/:orderNo/pay-deposit - 标记定金已付
PUT /api/agreement/:orderNo/pay-balance - 标记尾款已付
```

## 🌐 多语言支持详解

### 实现方式：客户端翻译

**原理：**
```javascript
const translations = {
  zh: {
    pageTitle: '设计服务协议',
    // ... 所有中文文本
  },
  en: {
    pageTitle: 'Design Service Agreement',
    // ... 所有英文文本
  }
};

function switchLanguage() {
  currentLang = currentLang === 'zh' ? 'en' : 'zh';
  const t = translations[currentLang];
  
  // 更新所有文本
  document.getElementById('pageTitle').textContent = t.pageTitle;
  // ...
}
```

**优点：**
- ✅ 无需刷新页面
- ✅ 切换即时响应
- ✅ 不影响用户数据

## 🔐 数据安全

### 当前方案（localStorage）

**安全级别：中等**

**保护措施：**
- ✅ 仅存储基本信息
- ✅ 不存储密码
- ✅ 不存储支付信息
- ✅ 本地存储，不上传

**风险：**
