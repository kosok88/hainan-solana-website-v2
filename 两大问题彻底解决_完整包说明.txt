- 新链接：`<a href="index.html#policy">政策</a>`
- 新链接：`<a href="index.html#news">新闻</a>`

**效果：**
点击"政策"或"新闻"会跳转到主页对应位置 ✅

## 💡 高级解决方案

### 如果上传了所有文件，Vercel还是空白：

#### 方案1：手动触发重新部署

1. 登录 Vercel
2. 进入项目
3. Deployments → ... → Redeploy
4. 选择 "Redeploy without Cache"
5. 等待部署完成

#### 方案2：检查项目设置

Vercel项目设置：
- Production Branch: main 或 master ✅
- Root Directory: 空 或 / ✅
- Framework Preset: Other 或 空 ✅
- Build Command: 空 ✅

#### 方案3：重新连接

1. Settings → Git → Disconnect
2. 删除项目
3. 重新 Import from GitHub
4. 选择仓库
5. Deploy
