## 3. 👤 4个人物图片显示调整

### 问题人物：
- p7.jpg  - 王文明（头像不在中间）
- p8.jpg  - 冯白驹（嘴巴下巴看不见）
- p9.jpg  - 邢诒孔（头像位置偏下）
- p11.jpg - 陈峰（显示位置不佳）

### 原因分析：
这4张都是证件照类型：
- 王文明：194x155（横向小图，局部）
- 冯白驹：400x609（竖版）
- 邢诒孔：400x515（竖版）
- 陈峰：255x296（竖版证件照）

### 解决方案：
为这4张图片添加特殊CSS规则：
```css
img[src="p7.jpg"],
img[src="p8.jpg"],
img[src="p9.jpg"],
img[src="p11.jpg"] {
    object-position: center top;
}
```

### 效果：
**修改前：**
- 显示中间部分 → 看不到头部
- 看到身体、衣服 → 看不到脸

**修改后：**
- 从顶部中间开始显示
- 优先显示头部和脸部
- 能看到完整的头像

### 显示对比：

## 📊 技术细节

### LOGO圆形裁剪原理
```css
border-radius: 50%;    /* 变成圆形 */
object-fit: cover;     /* 填充满圆形区域 */
width: 120px;          /* 宽度 */
height: 120px;         /* 高度（自动匹配） */
```

### 背景图定位
```css
background: 
  linear-gradient(蒙版),    /* 第1层：紫色蒙版 */
  url('sc1.jpg');           /* 第2层：亚龙湾图片 */
background-size: cover;     /* 覆盖整个区域 */
background-position: center; /* 居中显示 */
background-attachment: fixed; /* 固定背景（视差） */
```

### 人物图片定位
```css
/* 默认：所有人物 */
object-position: top center;  /* 从顶部中间开始 */

/* 特殊：4个证件照 */
object-position: center top;  /* 从中间顶部开始 */
```
