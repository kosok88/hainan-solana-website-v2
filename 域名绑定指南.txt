## 📋 绑定步骤

### 步骤1：登录Vercel后台

1. **访问 Vercel**
   ```
   https://vercel.com
   ```

2. **登录账号**

3. **进入项目**
   ```
   找到：hainan-solana-website-v2
   点击进入
   ```

---

### 步骤2：添加域名

1. **点击 "Settings"（设置）**
   ```
   在项目页面顶部导航
   ```

2. **点击左侧 "Domains"（域名）**

3. **输入您的域名**
   ```
   在输入框输入：solana-hainan.com
   点击 "Add"
   ```

4. **Vercel会显示DNS设置要求**

---

### 步骤3：配置DNS解析

Vercel会要求您添加以下DNS记录：

#### 方法A：使用A记录（推荐）⭐⭐⭐

```
类型：A
名称：@
值：76.76.21.21
TTL：自动或3600
```

```
类型：A  
名称：www
值：76.76.21.21
TTL：自动或3600
```

---

#### 方法B：使用CNAME记录

```
类型：CNAME
名称：@
值：cname.vercel-dns.com
TTL：自动或3600
```

```
类型：CNAME
名称：www  
值：cname.vercel-dns.com
TTL：自动或3600
```

---

### 步骤4：在域名注册商处添加DNS记录

**如果域名在阿里云：**

1. 登录阿里云控制台
2. 进入"域名"管理
3. 点击"解析"
4. 添加上面的记录

**如果域名在腾讯云：**

1. 登录腾讯云控制台
2. 进入"我的域名"
3. 点击"解析"
4. 添加上面的记录

**如果域名在GoDaddy/Namecheap等：**

1. 登录域名管理后台
2. 找到DNS管理
3. 添加A记录或CNAME记录

---

### 步骤5：等待DNS生效

- **生效时间：** 10分钟 - 48小时
- **通常：** 1-2小时内生效
- **检查方法：** 访问 solana-hainan.com

---

### 步骤6：启用HTTPS（自动）

Vercel会自动：
1. 检测DNS配置
2. 申请SSL证书
3. 启用HTTPS
4. 强制跳转HTTPS

**通常5-10分钟完成**

## 🔧 常见问题

### Q1: DNS多久生效？

A: 通常1-2小时，最长48小时。可以用无痕窗口测试。

---

### Q2: 需要删除旧的DNS记录吗？

A: 如果有其他A记录或CNAME记录指向其他地方，需要先删除。

---

### Q3: www和不带www都能访问吗？

A: 是的，两个都配置了，Vercel会自动处理跳转。

---

### Q4: HTTP会自动跳转HTTPS吗？

A: 是的，Vercel会自动强制HTTPS。

---

### Q5: 需要备案吗？

A: 
- 域名在国内注册商：可能需要
- Vercel服务器在国外：不一定需要
- 建议咨询域名注册商
