# 🚀 13个园区系统 - 安装指南

## ⚡ 最简单的方法（推荐）

由于您的 index.html 文件已经很完善，我们只需要添加一小段代码即可！

---

## 📝 安装步骤

### 第1步：上传 parks_complete_system.js

1. 登录 https://github.com/kosok88/hainan-solana-website-v2
2. 点击 "Upload files"
3. 上传 `parks_complete_system.js`
4. 点击 "Commit changes"

### 第2步：修改 index.html

**打开您的 index.html 文件，找到最后的 `</body>` 标签**

在 `</body>` **之前**，复制粘贴 `add_parks_modal.txt` 文件里的全部内容

#### 具体位置：

```html
... 您的浮动按钮代码 ...
</script>

👉 在这里粘贴 add_parks_modal.txt 的内容 👈

</body>  ← 这是 body 结束标签
</html>
```

### 第3步：修改园区入驻服务卡片

找到这一行：

```html
<a href="services.html#service12" class="card" ...>
<h3 id="s9">🏢 园区入驻服务</h3>
```

**改成：**

```html
<div class="card" onclick="showParkDetail('park01')" style="cursor:pointer;transition:.3s;" onmouseover="this.style.transform='translateY(-5px)';this.style.boxShadow='0 10px 25px rgba(5,150,105,.2)'" onmouseout="this.style.transform='translateY(0)';this.style.boxShadow=''">
<h3 id="s9">🏢 园区入驻服务</h3>
<p id="s9d">海南各大园区对接、入驻条件咨询、政策优惠申请、一站式服务</p>
<div style="text-align:center;margin-top:15px;color:#059669;font-weight:600;font-size:14px;">
  点击查看13个园区详情 →
</div>
</div>
```

### 第4步：上传并测试

1. 保存修改后的 index.html
2. 上传到 GitHub（覆盖原文件）
3. 等待 1-2 分钟
4. 访问 www.solana-hainan.com
5. 点击"园区入驻服务"测试
