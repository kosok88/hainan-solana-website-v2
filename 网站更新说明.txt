### 3. 📝 会员注册资料说明
**注册资料在哪里？**

目前网站是**前端展示网站**，没有后端数据库。会员注册有两种方式：

#### 方式1：扫码注册（当前方式）
用户扫描微信/Telegram二维码 → 发送"注册会员" → 客服手动记录

**优点：** 简单快速，无需开发
**缺点：** 需要手动管理

#### 方式2：表单提交（可升级）
用户填写表单 → 数据发送到您的邮箱/数据库

**需要配置：**
1. 后端服务（Node.js/PHP）
2. 数据库（MySQL/MongoDB）
3. 或使用第三方表单服务（如Google Forms、金数据）

**当前建议：** 
保持现有扫码方式，简单有效。如果需要自动化收集，可以：
- 使用Google Forms嵌入网站
- 使用腾讯问卷
- 或联系我配置后端服务

#### 如何添加真实视频？

**当前状态：** 使用YouTube示例链接（需要替换）

**替换步骤：**

1. **准备视频链接**
   - YouTube视频：复制视频链接
   - B站视频：复制嵌入代码
   - 腾讯视频：复制嵌入代码

2. **获取嵌入链接**
   
   **YouTube示例：**
   ```
   原链接：https://www.youtube.com/watch?v=ABC123
   嵌入链接：https://www.youtube.com/embed/ABC123
   ```

   **B站示例：**
   ```
   点击分享 → 嵌入代码 → 复制iframe的src
   ```

3. **替换HTML中的链接**
   
   找到index.html中的这6行：
   ```html
   <iframe src="https://www.youtube.com/embed/dQw4w9WgXcQ" ...>
   ```
   
   替换为你的实际视频链接

4. **示例替换**
   ```html
   <!-- 原来 -->
   <iframe src="https://www.youtube.com/embed/dQw4w9WgXcQ" ...>
   
   <!-- 替换为 -->
   <iframe src="https://www.youtube.com/embed/你的视频ID" ...>
   ```
