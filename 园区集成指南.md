## 第3步：为每个园区添加点击事件

在您现有的13个园区HTML代码中，为每个园区添加 `onclick` 事件。

**示例修改：**

### 修改前：
```html
<div class="park-item">
  <div class="park-header">
    <span class="park-number">1️⃣</span>
    <h3>海口江东新区</h3>
  </div>

  <div class="park-industry">金融服务、总部经济...</div>
</div>
```

### 修改后（添加点击事件）：
```html
<div class="park-item" onclick="showParkDetail('park01')" style="cursor:pointer;transition:.3s;" onmouseover="this.style.transform='translateY(-5px)';this.style.boxShadow='0 10px 25px rgba(5,150,105,.2)'" onmouseout="this.style.transform='translateY(0)';this.style.boxShadow=''">
  <div class="park-header">
    <span class="park-number">1️⃣</span>
    <h3>海口江东新区</h3>
  </div>
  <div class="park-position">海南自贸港重点园区</div>
  <div class="park-policy">15%企业所得税、零关税...</div>
  <div class="park-industry">金融服务、总部经济...</div>
  <div style="text-align:center;margin-top:15px;color:#059669;font-weight:600;font-size:14px;">
    点击查看详情 →
  </div>
</div>
```
