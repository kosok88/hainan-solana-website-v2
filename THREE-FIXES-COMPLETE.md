## 🎬 动画效果

### 服务卡片悬停动画：

```
默认状态：
• 位置：normal
• 顶部线条：隐藏（scaleX: 0）
• 阴影：柔和黑色
• 边框：淡白色

悬停状态（0.4s过渡）：
• 位置：上浮8px ↑
• 顶部线条：展开（scaleX: 1）
• 阴影：绿色发光
• 边框：绿色高亮
• 背景：更透明
```

### CSS实现：

```css
.card:before {
  content: '';
  position: absolute;
  top: 0;
  height: 4px;
  background: linear-gradient(90deg, #4ade80, #22c55e);
  transform: scaleX(0);  /* 初始隐藏 */
  transition: .4s;
}

.card:hover:before {
  transform: scaleX(1);  /* 悬停展开 */
}

.card:hover {
  transform: translateY(-8px);  /* 上浮 */
  box-shadow: 0 12px 40px rgba(74,222,128,.3);  /* 绿光 */
}
```
