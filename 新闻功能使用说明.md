# 📰 新闻功能使用说明

## 🎯 功能特点

### 1. 显示最新5条新闻
- 首页默认显示最新5条新闻
- 自动按日期倒序排列
- 中英文双语切换

### 2. 可展开查看更多
- 点击"查看更多"按钮展开旧新闻
- 点击"收起"按钮折叠旧新闻
- 平滑动画效果

### 3. 数据分离管理
- 所有新闻数据在 `news_data.js` 文件中
- 方便添加、修改、删除新闻
- 无需修改HTML代码

---

## 📝 如何添加新闻

### 步骤1：打开 news_data.js

找到这个文件，用文本编辑器打开

### 步骤2：在数组最前面添加新闻

```javascript
const newsData = [
  // 在这里添加新新闻（放最前面）
  {
    id: 'news_20251220',  // 唯一ID
    date: '2025-12-20',    // 日期
    title_zh: '🎉 新政策标题',  // 中文标题
    title_en: '🎉 New Policy',   // 英文标题
    content_zh: '这是中文内容描述...',  // 中文内容
    content_en: 'This is English content...',  // 英文内容
    tags_zh: ['标签1', '标签2'],  // 中文标签
    tags_en: ['Tag1', 'Tag2']      // 英文标签
  },
  // 旧新闻自动往下排
  ...
];
```

### 步骤3：上传文件

只需上传修改后的 `news_data.js` 文件到GitHub

### 步骤4：完成！

等待1-2分钟，网站自动更新

## ✅ 新闻数据格式说明

### 必填字段

| 字段 | 说明 | 示例 |
|------|------|------|
| id | 唯一标识符 | 'news_20251218' |
| date | 日期（YYYY-MM-DD） | '2025-12-18' |
| title_zh | 中文标题 | '🎉 海南封关启动' |
| title_en | 英文标题 | '🎉 Hainan FTP Launch' |
| content_zh | 中文内容（100-200字） | '...' |
| content_en | 英文内容（100-200字） | '...' |
| tags_zh | 中文标签数组 | ['政策', '封关'] |
| tags_en | 英文标签数组 | ['Policy', 'Closure'] |

### 注意事项

1. **ID命名规则：** news_YYYYMMDD（日期格式）
2. **日期格式：** 必须是 YYYY-MM-DD
3. **标题：** 建议加Emoji，更醒目
4. **内容：** 100-200字为佳，简明扼要
5. **标签：** 2-3个为宜

## 🔄 如何删除旧新闻

### 方法1：直接删除
打开 news_data.js，删除对应的新闻对象

### 方法2：暂时隐藏
把不想显示的新闻移到数组末尾

### 建议
- 保留最近6-12个月的新闻
- 太旧的新闻可以删除

## 💡 使用技巧

### 技巧1：重要新闻置顶
把最重要的新闻放在数组第一个位置

### 技巧2：使用醒目的Emoji
- 政策：🎯 📋 ⚖️
- 经济：💰 📈 💼
- 发展：🚀 🌟 ⭐
- 庆祝：🎉 🎊 🥳

### 技巧3：标签分类
- 政策类：政策更新、法规变动
- 经济类：贸易数据、投资信息
- 发展类：项目进展、成就展示

## 🔧 故障排查

### 问题1：新闻不显示
- 检查 news_data.js 是否正确上传
- 检查日期格式是否正确（YYYY-MM-DD）
- 检查是否有语法错误（逗号、引号）

### 问题2：中英文切换不工作
- 确保两个语言字段都已填写
- 刷新页面（Ctrl+F5）

### 问题3："查看更多"不显示
- 当前只有5条或更少新闻时，不显示按钮
- 添加第6条新闻后自动显示
