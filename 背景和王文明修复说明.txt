# 🔧 背景图和王文明修复说明

## ✅ 本次修复内容

### 1. 主页背景图修复
**问题：** 上传了sc1.jpg，但背景还是没变

**原因：** 浏览器强缓存了旧的背景图

**解决方案：**
```css
/* 修改前 */
background: url('sc1.jpg');

/* 修改后 */
background: url('sc1.jpg?v=2');  /* 添加版本号 */
```

**原理：**
- `?v=2` 让浏览器认为这是新文件
- 强制重新加载，不使用缓存
- 这是缓存破坏（cache busting）技术

---

### 2. 王文明图片显示修复
**问题：** 其他3个人物（冯白驹、邢诒孔、陈峰）正确了，但王文明还不对

**原因：** 王文明的图片特殊
- 尺寸：194x155（横向小图）
- 与其他人物的竖版图不同
- 需要不同的显示策略

**解决方案：**
```css
/* 冯白驹、邢诒孔、陈峰（竖版图） */
.card img[src="p8.jpg"],
.card img[src="p9.jpg"],
.card img[src="p11.jpg"] {
    object-position: center !important;  /* 居中 */
}

/* 王文明（横版小图）- 特殊处理 */
.card img[src="p7.jpg"] {
    object-position: top !important;     /* 从顶部开始 */
    object-fit: contain !important;      /* 完整显示，不裁剪 */
    background: #f0f0f0;                 /* 浅灰背景 */
}
```

**效果：**
- 王文明图片完整显示（不裁剪）
- 从顶部开始显示
- 周围有浅灰色背景填充

## 📊 技术细节

### 缓存破坏（Cache Busting）
```css
/* 方法1：版本号 */
url('sc1.jpg?v=2')

/* 方法2：时间戳 */
url('sc1.jpg?t=20251220')

/* 方法3：哈希值 */
url('sc1.jpg?h=abc123')
```

任何一种都能破坏缓存！

### Object-fit 对比
```css
/* cover - 填充裁剪（可能切掉部分） */
object-fit: cover;
→ 填满整个框，多余部分被切掉

/* contain - 完整显示（可能有空白） */
object-fit: contain;
→ 完整显示图片，不裁剪，周围可能有空白
```

王文明是横向小图，用contain完整显示最好！
