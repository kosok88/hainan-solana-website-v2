# 🔧 图片显示修复说明

## ✅ 已修复的问题

### 问题1：人物图片显示不全
**症状：** 丘濬、冼夫人、苏东坡三个人物只看到身体，看不到头部

**原因：** 
- 这三张图片是竖版长图（高度远大于宽度）
- 卡片固定高度200px
- 默认居中裁剪导致显示中间部分（身体）而不是顶部（头部）

**解决方案：**
```css
/* 修改前 */
object-fit: cover;

/* 修改后 */
object-fit: cover;
object-position: top center;  ← 从顶部开始显示
```

**效果：**
✅ 现在所有人物都能看到头部
✅ 优先显示头部和上半身
✅ 保持图片美观不变形

---

### 问题2：服务部分文字清晰度不够
**症状：** 服务栏目的文字不够清晰，对比度不足

**解决方案：**
1. **增强背景对比度**
   ```css
   /* 修改前 */
   background: rgba(0,0,0,.25);
   
   /* 修改后 */
   background: rgba(0,0,0,.4);  ← 更深的背景
   ```

2. **增强标题**
   ```css
   font-size: 22px;         ← 从20px增大
   font-weight: 800;        ← 从700加粗
   text-shadow: 更强的阴影
   letter-spacing: 0.5px;   ← 增加字间距
   ```

3. **增强内容文字**
   ```css
   font-size: 18px;         ← 从17px增大
   font-weight: 600;        ← 从500加粗
   text-shadow: 更强的阴影
   letter-spacing: 0.3px;   ← 增加字间距
   ```

**效果：**
✅ 文字更清晰易读
✅ 对比度更高
✅ 整体更专业

## 🔍 技术细节

### object-position 说明
```
top center  = 从上方中间开始显示
top left    = 从左上角开始显示
top right   = 从右上角开始显示
center      = 从中心开始显示（默认）
bottom      = 从底部开始显示
```

对于竖版人物画像，使用 `top center` 最合适！

### 文字清晰度优化原理
1. **增加字号** - 更容易阅读
2. **加粗字重** - 更醒目
3. **深色背景** - 提高对比度
4. **强化阴影** - 增强立体感
5. **字间距** - 减少拥挤感
